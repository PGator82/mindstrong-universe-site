① Hook

The Price Tag Without a Price

You're at a store. T-shirts cost some unknown amount. You grab 3 of them and a $5 bag. What's the total?

You can't give a number — but you can write the relationship: 3t + 5

t
+
5

That's an algebraic expression. It's not an equation yet — no equals sign. It's just a description of a situation. And once you know t, you can calculate everything.

② The Hack

Variables Are Boxes. Terms Are Chunks.

Every expression has three kinds of parts. Once you can name them, you can manipulate them.

⚡ The Hack — Anatomy of an Expression
Coefficient = the number in front of a variable (the multiplier)
Variable = the unknown box (x, n, t, etc.)
Constant = a plain number, no variable attached
Term = any single chunk (coefficient + variable OR just a constant)
5x
Term: coeff 5, var x
-3n

Combining Like Terms: Only combine terms with the same variable.
3x + 5x = 8x ✅ — same variable
3x + 5y ≠ 8xy ❌ — different variables, can't combine
2x + 4 + x + 3 = 3x + 7 ✅ — collect x's, collect constants

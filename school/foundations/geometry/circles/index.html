① Hook

The Number That Obsessed Humanity

Mathematicians have been calculating π for over 4,000 years. Egyptians approximated it as 3.16. Archimedes got it to 3.14. Today we know it to trillions of decimal places — and it never repeats.

Every circle you've ever seen — wheels, coins, planets — shares the same secret: if you walk around the outside and then measure across, the ratio is always π.

② The Hack

Two Formulas. Same π.

Everything about circles comes from one constant: π ≈ 3.14 (or 22/7 as a fraction). Memorize two formulas — done.

⚡ Circle Formulas
Circumference = 2πr = πd
Area = πr²

r = radius, d = diameter = 2r
Use π ≈ 3.14 for calculations

Circumference
C = 2πr
C = πd

Area
A = πr²

r=5 → A = 3.14×5² = 3.14×25 = 78.5

Note: answers using π≈3.14 may differ slightly from exact π. Both accepted in practice — use 3.14 unless told otherwise.
